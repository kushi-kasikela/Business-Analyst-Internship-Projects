Business Analyst Internship Projects



This repository showcases four Excel-based dashboard projects I completed as part of my Business Analyst internship.
Each project demonstrates my ability to analyze raw data, apply business logic, design KPIs, and build insightful dashboards using Excel.

These projects highlight not just technical skills, but also analytical thinking ‚Äî transforming data into clear business insights and visual stories.



  Project 1: Financial Performance Dashboard

Goal: Analyze company sales, profit, and discount trends to evaluate overall financial performance.
Key Insights: Identified best-performing regions, impact of discounts on profit, and year-on-year growth patterns.
Tools Used: Excel (Pivot Tables, Charts, Slicers)



  Project 2: Stock Market Analysis Dashboard

Goal: Study daily stock movements and calculate daily returns to visualize market performance over time.
Key Insights: Revealed stock volatility, upward trends, and relative performance among multiple companies.
Tools Used: Excel (Formulas, Line Charts, Data Visualization)



  Project 3: Supply Chain Management Dashboard

Goal: Evaluate supplier efficiency, shipping costs, and lead times to optimize the supply chain process.
Key Insights: Highlighted reliable suppliers, efficient shipping carriers, and cost-performance trade-offs.
Tools Used: Excel (Pivot Tables, KPIs, Slicers, Interactive Dashboards)



  Project 4: Top Instagram Influencers Dashboard

Goal: Analyze influencer engagement, reach, and growth trends across countries.
Key Insights: Identified top-performing influencers, audience engagement patterns, and emerging markets for influencer growth.
Tools Used: Excel (Data Cleaning, Advanced Formulas, Charts, Interactive Dashboards)
